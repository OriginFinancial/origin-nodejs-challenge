# Origin Node.js Challenge

Node.js challenge for backend candidates

## Requirements

- Node.js >= v14.0
- Yarn >= v1.22

## Installation

In project's directory, run the following commands:

```
yarn install
```

## Running locally

To run the project locally with hot reload:

```
yarn dev
```

### Lint

To view lint issues:


```
yarn lint
```

To fix them:

```
yarn lint:fix
```
